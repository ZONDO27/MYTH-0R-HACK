# Myth Or Hack

## Overview
Myth Or Hack is an Android app designed to test popular myths and hacks interactively. It allows users to explore, verify, and share results in a fun and engaging way.

## Features
- Interactive myth testing
- User-friendly interface
- Real-time feedback and results
- Local data storage for experiments

## Testing & Automation
This project uses **GitHub Actions** to automate builds and tests, ensuring the app runs smoothly across environments.
